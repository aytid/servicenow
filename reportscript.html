Using Script Include in Filter – ServiceNow

To use a Script Include in a filter condition (like in reports or reference qualifiers), make sure the Script Include is marked as "Client Callable" and "Accessible from Sandbox".

Steps:

  • Go to the Script Include you want to use.
  • Check both Client Callable and Accessible from Sandbox.
  • Write your function and make sure it's global (or public static method).

Sample Script Include:

var MyUtils = Class.create();
MyUtils.prototype = Object.extendsObject(AbstractAjaxProcessor, {
  isManager: function(userId) {
    var gr = new GlideRecord('sys_user');
    gr.get(userId);
    return gr.title == "Manager";
  }
});

Using It in a Reference Qualifier:

javascript:MyUtils.isManager(current.requested_for);

When used like this, the filter will evaluate whether the user is a manager, using the Script Include logic.
